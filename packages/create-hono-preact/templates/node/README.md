# {{name}}

A [hono-preact](https://framework.sbesh.com) app, scaffolded for the Node.js adapter.

## Develop

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Start production server

```bash
pnpm start
```

The server listens on `PORT` (default 3000).
