# Monorepo Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Convert the hono-preact repo into a pnpm monorepo with the app at `apps/app` and framework code extracted into `packages/@hono-preact/*`.

**Architecture:** Phase 1 restructures the repo (pnpm + app move, no import changes). Phase 2 extracts framework packages one at a time, updating the app to consume each before proceeding. Each phase ends with a green build.

**Tech Stack:** pnpm workspaces, TypeScript (tsc), Vite, Cloudflare Workers (wrangler)

---

## File Map

### Phase 1 creates:
- `pnpm-workspace.yaml` — workspace package globs
- `package.json` (root) — private, delegate scripts only
- `tsconfig.json` (root) — shared base compiler options
- `apps/app/` — everything currently at root moves here
- `apps/app/package.json` — renamed from root, `"name": "app"`
- `apps/app/tsconfig.json` — extends `../../tsconfig.json`, app-specific overrides

### Phase 2 creates (one task each):
- `packages/vite/package.json`, `packages/vite/tsconfig.json`, `packages/vite/src/index.ts`
- `packages/iso/package.json`, `packages/iso/tsconfig.json`, `packages/iso/src/index.ts`, `packages/iso/src/*.ts(x)` (moved from `apps/app/src/iso/`)
- `packages/server/package.json`, `packages/server/tsconfig.json`, `packages/server/src/index.ts`, `packages/server/src/*.ts` (moved from `apps/app/src/server/context.ts` and `apps/app/src/server/middleware/`)
- `packages/hono-preact/package.json`, `packages/hono-preact/tsconfig.json`, `packages/hono-preact/src/index.ts`

### Phase 2 modifies:
- `apps/app/vite.config.ts` — update import of vite plugin
- `apps/app/src/iso/` files that import from `./page.tsx` etc. — unchanged (no `.ts` extension imports to fix)
- `apps/app/src/server/layout.tsx` — update import from `'./context.js'` to `'@hono-preact/server'`
- `apps/app/src/server.tsx` — update import from `'./server/middleware/location.js'` to `'@hono-preact/server'`
- `apps/app/package.json` — add `workspace:*` deps as each package is extracted
